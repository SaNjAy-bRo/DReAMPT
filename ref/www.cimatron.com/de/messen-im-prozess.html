    Software zur Einhaltung der Toleranzvorgaben

    KOSTENLOSE TESTVERSION ANFORDERN

    Die Software ermöglicht das Messen bestimmter Maße auf der CNC-Maschine und validiert die aktuellen Bearbeitungsergebnisse. Der Benutzer kann wählen, ob er eine automatische Feinabstimmung der aufeinanderfolgenden Bearbeitungen vornehmen möchte oder ob er die Maschine anhalten möchte, um Bearbeitungen außerhalb der Toleranz zu vermeiden.

    Messen im Prozess

    Kontrollieren Sie die gefertigten Werkstücke auf Maßhaltigkeit und korrigieren Sie direkt mit der Längen- oder Radiuskorrektur automatisch auf der Fräsmaschine.

    • Es stehen mehrere Messzyklen zur Verfügung z.B. Punkt, Tasche, Rechteck, Bohrung, Zylinder.
    • Definieren Sie passende Toleranzen für jeden Zyklus.
    • Alle Bewegungen sind Kollisionsgeprüft gegen das Teil und das Rohteil (Stock).
    • Verfügbar für mehrere Steuerungen z.B. Heidenhain, Röders.

    Preset

    Tasten Sie automatisch das Werkstück an und setzten den Nullpunkt (Bezugspunkt) auf der Fräsmaschine oder Messmaschine.

    • Unterstützung von Fräsmaschinen mit Steuerungen von Heidenhain, Röders, Siemens.
    • Unterstützung von Messmaschinen von Zeiss (Calypso), Hexagon, Wenzel, Mitutoyo.
    • Es stehen mehrere Zyklen zur Verfügung.
